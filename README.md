# Porsche Executive Sales Dashboard

Dashboard executivo premium inspirado na identidade visual e experiência de usuário da Porsche Brasil, desenvolvido para análise comercial e visualização estratégica de vendas automotivas de luxo.

O projeto utiliza HTML, TailwindCSS, JavaScript e Chart.js para criar uma experiência moderna, elegante e responsiva.

* Filtros dinâmicos

* Análise de faturamento e vendas

# Funcionalidades

## Filtros Interativos

* Modelo Porsche
* Ano do Modelo
* Cidade
* Forma de Pagamento

Os filtros atualizam automaticamente os gráficos e indicadores do dashboard.

---

# KPIs Executivos

* Faturamento Total
* Total de Vendas
* Ticket Médio
* Modelo Mais Vendido

---

# Análises de Negócio

* Modelos Porsche mais vendidos por cidade
* Faturamento por ano do modelo
* Receita por cidade
* Receita por forma de pagamento
* Insights estratégicos de mercado
